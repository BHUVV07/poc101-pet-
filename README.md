# 🐾 PawLuxury — Bespoke Pet Lifestyle & Care Ecosystem

**PawLuxury** (under Manasa Pet Shop) is a full-stack Next.js web application representing an integrated, operational network of physical pet care locations, specialty hospitals, and veterinary wholesale supply divisions across Shivamogga, Karnataka.

---

## 🏢 The PawLuxury Ecosystem

PawLuxury bridges premium digital commerce with our physical, real-world facilities:

1. **Premium Retail Outlets**: Boutique physical storefronts located at Garden Area & Police Chowki, Shivamogga, Karnataka.
2. **Buddy & Kitty Specialty Pet Hospital**: Elite care clinic located at 100 Ft Road, offering board-certified veterinary, pediatric, and surgical specialist consultations.
3. **Veterinary Wholesale Division**: Licensed apothecary supplies located at Kote.
4. **Multi-Brand Distribution (Petstep)**: Integrated distribution network located at GSKM Road.

---

## ✨ Features & Modules

- **Shop**: Bespoke pet retail storefront for premium foods, toys, and lifestyle items.
- **Consultation & Specialty Hospital**: Directly schedule consultations with board-certified veterinary practitioners.
- **Wholesale Portal**: Dedicated interface for retail partners and veterinary clinics to source apothecary and pharmaceutical supplies.
- **Blog & Education**: Professional pet care advice and articles.
- **State Management & Authentication**: Secure user authentication and cart/wishlist management synchronized with a Supabase database.

---

## 🛠️ Technology Stack

- **Framework**: Next.js 16.2.6 (using Turbopack) & React 19
- **Styling**: Tailwind CSS v4, PostCSS, Framer Motion (for premium micro-animations)
- **Icons**: Lucide React
- **State Management**: Zustand
- **Database/Backend Service**: Supabase (`@supabase/supabase-js`)
- **Language**: TypeScript & Node.js

## 🚀 Getting Started

### Prerequisites

Make sure you have Node.js installed (v18+ recommended).

### Installation

Install the project dependencies:
```bash
npm install
```

### Running the Development Server

Start the local development server with Turbopack:
```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser to see the application.

### Building for Production

Compile a production build of the application:
```bash
npm run build
```
